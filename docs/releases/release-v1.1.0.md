# Proxmox Clash 插件 v1.1.0 更新日志

**发布日期**: 2024-12-01  
**版本**: v1.1.0  
**代号**: Version Management Release

## 🎯 版本概述

v1.1.0 是一个功能增强版本，主要添加了版本管理功能和订阅管理功能，提升了插件的可维护性和用户体验。

## 🚀 新功能

### 版本管理功能
- 添加版本管理功能
- 支持自动检测和升级
- 新增版本比较和回滚功能
- 提供版本历史记录

### 订阅管理功能
- 新增订阅管理功能
- 支持订阅链接导入和更新
- 自动同步订阅配置
- 订阅状态监控

### 日志查看工具
- 添加日志查看工具
- 实时日志监控
- 日志级别过滤
- 日志导出功能
